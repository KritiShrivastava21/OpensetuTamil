ஆவணங்கள்

ஓபன் செது: இது எப்படி வேலை செய்கிறது

ஓபன் செது DNS over HTTPS (DOH) மூலம் இயக்கப்படுகிறது, இது HTTPS வழியாக டொமைன் பெயர்களை பாதுகாப்பாக தீர்க்கும் நெட்வொர்க் புரொடோகால் ஆகும். DOH பயனர் தனியுரிமை மற்றும் பாதுகாப்பை மேம்படுத்துகிறது, நடுநிலைவர்களை DNS கேள்விகளை கண்காணிக்கவோ அல்லது மாற்றவோ தடுக்கும்.

இந்தியாவின் இணையம்

இந்தியாவின் இணையம் Handshake என்ற திறந்த மூல, மையமில்லாத பெயரிடும் புரொடோகால் மூலம் உருவாக்கப்பட்டுள்ளது, இது இணையத்திற்கு மாற்று ரூட் ஸோன் ஆக சேவை செய்கிறது. இது цен்சார்ஷிப்- எதிர்ப்பு, மாற்றமற்ற டொமைன் தீர்வை வழங்குகிறது. இது நமது இணையத்தின் முதுகெலும்பு.

விரைவில், ENS (Ethereum Name Service) மற்றும் IPFS (InterPlanetary File System) போன்ற புரொடோகால்களை ஆதரிக்க திட்டமிடுகிறோம், இது மேலும் திறந்த மற்றும் இடையீடு செய்யக்கூடிய இணையத்தை வழங்கும்.

உங்கள் சொந்த DOH உருவாக்குங்கள்

பள்ளி, பல்கலைக்கழகம், ஸ்டார்ட்அப், இலாப நோக்கு இல்லாத நிறுவனம், அல்லது அரசு அமைப்பு எவரும் பாரதிய இணையத்தை அணுக மற்றும் விரிவாக்க DOH தீர்வாளர் உருவாக்கலாம். ஓபன் செது என்பது ஒரு எடுத்துக்காட்டு மட்டுமே; நீங்கள் உங்கள் சொந்தத்தை இயக்கலாம்.
